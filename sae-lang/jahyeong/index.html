세어/자형

세어의 한자 자형을 설명하는 문서입니다. 접속 환경에 따라 자형이 다르게 보일 수 있습니다. 이 때는 한자 옆의 #을 클릭하시기 바랍니다(단, 유니코드에서 분리된 한자 한정).

이전 문서로

한국 정자 세어 정체자 변화 응용/비고
靑 青 円→⺝ 淸(맑을 청)→清
情(뜻 정)→情 #
이 외 靑이 들어가는 한자 모두
通 通 # ⻍→⻌ 道(길 도)→道 #
進(나아갈 진)→進 #
이 외 ⻍이 들어가는 한자 모두
眞 真 # 匕→十, 𠃊→一 鎭(누를 진)→鎮
禛(복 받을 진)→禛 #
이 외 眞이 들어가는 한자 모두
神 神 # 示→礻 禮(예도 례)→礼
社(모일 사)→社 #
이 외 示가 왼쪽에(‘변’ 부수) 들어가는 한자 모두
數 数 婁→娄 樓(다락 루)→楼
僂(구부릴 루)→偻
이 외 婁가 들어가는 한자 모두
尙 尚 八→丷 說(말씀 설)→説
送(보낼 송)→送 #
平(평평할 평)→平 #
半(절반 반)→半 #
消(사라질 소)→消 #
勝(이길 승)→勝 #
益(더할 익)→益 #
曾(일찍 증)→曾 #
이 외에도 더 있지만 여기선 생략하겠습니다.
緖 緒 日 위의 점 삭제 者(사람 자)→者 #
諸(모두 제)→諸 #
이 외 者가 들어가는 한자 모두
壽 寿 자체 간략화 濤(물결 도)→涛
禱(빌 도)→祷
이 외 壽가 들어가는 한자 모두
爲 為 爫 간략화 僞(거짓 위)→偽
遙 遥 䍃→䍃 謠(노래 요)→謡 #
搖(흔들 요)→摇
이 외 䍃가 들어간 한자 모두
飮 飲 𩙿→飠 飯(밥 반)→飯 #
飼(기를 사)→飼 #
이 외 𩙿이 들어간 한자 모두
